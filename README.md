Paradox Drawer
===========

![alt text](https://raw.githubusercontent.com/filipmlynarski/paradox_drawer/master/paradox_3x3.png)

Installation
------------
```
git clone https://github.com/filipmlynarski/paradox_drawer.git
sudo apt-get install python3-tk
```

Usage
-----------
```
python3 main.py
```

### Configuration File ###
To modify figures follow instructions inside config file
